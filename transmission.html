TRANSMISSION PART

Transmission part is the part of telecommunication which is responsible
for the communication between towers.

AS we have seen on the RAN part, The BTS is transceiver, means that
it has the ability to send and receive signals. This is the same as Transmission part.

Actually, this part begins where the RAN part ends at. This part begins from the component
called Baseband.the main component of this section isAMM(Access Module Magazine).
Sometimes called TN(Traffic Node).


AMM is a container of other node cards such as MMU, NPU, LTU, RMM, ...
its use is to hold them as a single unity and powering them.

NPU (Node Processing Unity)

This is the main node within TN, it is used to connect RAN and transmission via TR ports.
it also holds the card called RMM (Removable Memory Module) which stores all configuration data of the BTS.
The interface between NPU and Baseband is Ethernet cables.



MMU (Modem Module Unity)

This is the component which is responsible for modulation and demodulation of signals.
It also acts as an interface between TN and RAU(Radio Access Unity), and via IF cable, RAU is powered by MMU.

RAU (Radio Access Unity)

This component is used to convert sent signals from Electrical form to Microwave frequencies.
It also convert received signals from microwave frequencies to electrical form.



Microwave Antenna

This component is used for the communication between towers in contact.
It is responsible for transmitting and receiving frequencies between two or more towers in contact.
